### vue-router 설정
- vue router는 vue.js를 이용하여 싱글 페이지 애플리케이션(SPA)를 구현할 때 사용하는 vue.js의 공식 라우터
- 라우터라고 하면 일반적으로 네트워크간에 데이터를 전송하는 장치를 말한다. 뷰에서 말하는 라우터는 쉽게 말해서 URL에 따라 어떤 페이지를 보여줄지 매핑해주는 라이브러리
- 설치
```shell
npm install vue-router
```
